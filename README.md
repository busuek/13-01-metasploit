### Задание 2

Проведите сканирование Metasploitable в режимах SYN, FIN, Xmas, UDP.

Запишите сеансы сканирования в Wireshark.

Ответьте на следующие вопросы:

**Чем отличаются эти режимы сканирования с точки зрения сетевого трафика?**

1. **SYN-сканирование (-sS):** Nmap отправляет TCP-пакет с флагом SYN, открытый порт отвечает SYN/ACK, закрытый - RST/ACK.

2. **FIN-сканирование (-sF):** Отправляется TCP-пакет с флагом FIN, открытые порты игнорируют такие пакеты, закрытые отвечают RST.

3. **Xmas-сканирование (-sX):** Аналогично FIN, но отправляется пакет с флагами FIN, PSH и URG.

4. **UDP-сканирование (-sU):** Отправляются пустые UDP-пакеты. Закрытые порты отвечают ICMP "Port Unreachable", открытые либо не отвечают, либо отправляют данные.


**Как отвечает сервер?**

Сервер корректно реагирует на все типы сканирования: на SYN-сканирование отправляет SYN/ACK для открытых портов, на FIN/Xmas-сканирование игнорирует пакеты для открытых портов, а на UDP-сканирование возвращает ICMP-ошибки для закрытых UDP-портов.
